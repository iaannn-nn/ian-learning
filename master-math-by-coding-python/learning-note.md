## set up python virtual evironment. 

1.  why create python virtual environment: 
- to manage and reproduce the project easily without conflicting or missing dependency. 
A Python Virtual Environment is an isolated space where you can work on your Python projects, separately from your system-installed Python.

You can set up your own libraries and dependencies without affecting the system Python.
After you’ve learned to work with virtual environments, you’ll know how to help other programmers reproduce your development setup, and you’ll make sure that your projects never cause dependency conflicts for one another.

2. Create python virtual environment:
python -m venv /path/to/new/virtual/environment:
=> python -m venv learning. 
official doc: https://docs.python.org/3/library/venv.html.
real python: https://realpython.com/python-virtual-environments-a-primer/ 

using tutorial from real python. 

3. activate python virtual environment( learning):
- in (master math...) folder: source learning/bin/activate

4. install package in virtual env:
python -m pip install <package-name>

5. Deactivate It:
deactivate
